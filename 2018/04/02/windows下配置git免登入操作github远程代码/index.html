git 添加多个ssh key

需要场景：本地仓库添加了公司的gitlab或者github账号,你还想添加自己的github账号

生成一个新账号的ssh-key
ssh-key默认保存文件为id_rsa,新的ssh-key不能和默认的相同
ssh-keygen -t rsa -b 4096 -C “your_email@example.com“ -f ~/.ssh/xxx_rsa

新密钥添加到SSH agent中
默认只会读取id_rsa，为了让SSH识别新的私钥，需将其添加到SSH agent中
ssh-add ~/.ssh/xxx_rsa
如果出现Could not open a connection to your authentication agent的错误，就试着用以下命令
ssh-agent bash
ssh-add ~/.ssh/xxx_rsa

修改config文件
在~/.ssh目录下找到config文件，如果没有就创建
touch config # 创建config文件
