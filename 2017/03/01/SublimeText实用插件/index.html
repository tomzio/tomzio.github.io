SublimeText实用插件

发表于 2017-03-01 | 分类于 开发工具与环境

Sulbime Text3 常用插件整理

由于国内部分地区sublime.wbond.net域名被墙了，但通过ip可以访问先 Ping sublime.wbond.net 得到具体的IP地址

修改hosts文件 , 增加下面一行 50.116.34.243 sublime.wbond.net

####Sulbime Text3 Settings – User配置

####以下是比较实用的插件列表:

  1. Emmet
    提高HTML & CSS3编写速度。

  2. Sublime-HTMLPrettif
    HTML, CSS, JavaScript and JSON code formatter

  3. Jade
    Node.js 的 HTML 模板引擎

  4. Theme – Soda
    一直用的一款皮肤

  5. sideBarEnhancements
    侧边栏右键增效插件，提高页面处理速度。

  6. Color Highlighter

  7. SCSS

  8. Autoprefixer

  9. HTML5
    Sublime Text3支持HTML5

  10. Sass
    Sublime Text3支持Sass

  11. Less
    Sublime Text3支持Less

  12. SublimeCodeIntel
    代码提示工具

  13. jQuery
    Sublime Text3支持jQuery

  14. DocBlockr
    快速注释

  15. ConvertToUTF8
    UTF8转换

  16. FileDiffs
    查找文档不同

  17. Alignment
    等号对齐 快捷键会有冲突 重新在Alignment > Key Bindings – User 配置快捷键 ctrl+alt+f

  18. SublimeTmpl
    快速格式化代码

  19. Gutter Color
    颜色提示 需要imageMagick支持

  20. Can I Use
    可以直接调整到caniuse看到当前属性的浏览器支持情况。快捷键

  21. InsertDate
    插入时间，项目文件存在CDN的时候，需要改变时间戳才会看到效果，这个是个好东西。

  22. Git
    Git插件，提供Git常用的命令集合。

  23. Auto​File​Name
    自动提示路径插件

  24. WordCount
    可以实时显示当前文件的字数。

  25. OmniMarkupPreviewer
    可以实时显示当前文件的字数。
